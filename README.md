# NegativeEffectsMod

Places a lot of effects onto the Player throughout their journey. 

- Daedicar's Woe
- Shabriri's Woe
- Normal Poison
- Baldachin's Blessing
- Winged Greathorn
- -15% PHYSICAL DMG from Greyoll's Roar
- Lulling Branch Sleep Build Up
- Talisman of All Crucible Debuff
- Beloved Stardust Debuff
- -15% Healing
- Gideon Debuff
- Normal Scarlet Rot
- DS3 Maggots after a certain point
- Lake of Rot Scarlet Rot after a certain point


## Special Effects that are on intervals that the user can changed:
- Slow Down the Player where they are now heavy rolling and can't run
- Disabling Torrent & Frenzy Build Up
- Spirit Eel Stamina Drain
- Disable Grace Warping
- Death Blight Build Up
- No Healing
- Ruptured Tear Explosion
- Fake Death

**All These intervals can be manipulated by editing `player_event_flags.json` under the `data` folder**
